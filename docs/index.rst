Global Portfolio Monitoring's Upload Assistant package
======================================================
This application is an GUI ETL audit tool that helps manage the quality of data submitted by analysts via the GPM Data Collection Excel template.

This current and only version is for the Data Collect request planned for November 2020 that covers the reporting period of October 1st, 2019 through September 31st 2020.

Analyst end users are provided immediate report views, validations, and opportunities to note and provide feedback.


"README.txt" provides basic end user instructions. Initialize the application by running "main.py".
