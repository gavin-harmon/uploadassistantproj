The Upload Assistant

Custom data collection software created with pyinstaller and in production from 2019Q4 to 2021Q1.
Note, this is a view of dummy data used to test validations. Any nominal figure or ratio is obfusacted with random coefficients.

An initial report is displayed of the input as loaded.
This "PandasTable" interface is exportable and can even be used to create pivots and graphs in the application.

There are 26 highly custom validation screens that only display if there is an issue in the submission. 1/2

Second example of a validation screen. 2/2

After validations are navigated, there is a full report. If this were a real user, they user would be encouraged to resubmit.















Once the user validates the data is the best available to them, the program converts the final version to json and parquet format for pipeline ingestion and also produces financial reporting the user can keep as reciepts. Screenshots to be added.
