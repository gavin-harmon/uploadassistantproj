The Upload Assistant.

Custom data software I built and put into production in 2019.   This system vastly improved data quality vs systems administered by much larger departments. It was ended when the department head and scientist started a job at a new company,   Please note this is a view of dummy data, meant to test validations. Everything in the below screenshots contains altered data meant to obfuscate any nominal figure or ratio.

 

An initial report of the input as loaded. This "Pandastable" interface is exportable and can even be used to do pivots and graphing.

 

There are 26 highly custom validation screens that only pop up if there is an issue in the submission. .

 

After validations are navigated, there is a full report. This user would be encouraged to resubmit..

Once we have the cleanest data available, the program converts the final version to json format for pipeline ingestion and also produces financial reporting the user can keep as reciepts.
